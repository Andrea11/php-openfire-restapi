Php-openfire-restapi

Remotely manage the Openfire server using Rest Api

View project on GitHub

Description.

This class can manage an Openfire real time collaboration (RTC) server calling its REST API. It can send HTTP requests the Web server of a given Openfire server to perform several management operations. Currently it can get the list of Openfire users in the server, get the details of an user, add, remove, update, lockout and unlock users, add, remove and update users in a roster.
